About This Repository

Natural Language Processing (NLP) allows computers to understand and work with human language. Before a computer can understand text, the text must be cleaned, organized, and converted into a format that machines can process.


This repository focuses only on the Week 1 syllabus. Every section has been written in simple English with examples and diagrams so that students can quickly understand the main ideas without reading lengthy theory.
